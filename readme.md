# Lumen + Bootstrap Sass + Font Awesome Sass Boilerplate

## Requirements


  - Composer installed globally or composer.phar
  - Sass installed globally
  - Compass installed globally


## Installation

```sh
$ git clone [git-repo-url] my-app
$ cd my-app
$ composer install
```

For compass
```sh
$ cd my-app/public/assets
$ compass watch
```
